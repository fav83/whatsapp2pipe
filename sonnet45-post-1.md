## Step-by-Step Setup

Chat2Deal requires minimal configuration. If you have Pipedrive and use WhatsApp Web, you're ready to start.

### Install the Extension

1. Visit the Chrome Web Store and search for "Chat2Deal" (or use the direct link from [chat2deal.com](https://chat2deal.com))
2. Click "Add to Chrome"
3. Confirm the installation when prompted

The extension requests permission to access WhatsApp Web (to read contact information) and to connect to Pipedrive (to sync contacts). It does not read your WhatsApp message content or store conversation history.

### Sign In with Pipedrive

1. Open WhatsApp Web in Chrome
2. You'll see a new sidebar panel on the right side of the screen
3. Click "Sign in with Pipedrive" in the sidebar
4. A popup window opens with Pipedrive's OAuth authorization screen
5. Review the permissions (Chat2Deal needs access to read and create contacts) and click "Allow"
6. The popup closes automatically and you're signed in

The entire process takes about 60 seconds. No API keys to configure, no IT team required, no complex settings to manage.

### Grant Necessary Permissions

During sign-in, Pipedrive will ask you to authorize Chat2Deal to access your contacts. This is standard OAuthâ€”the same secure authentication method used by major apps across the web.

Chat2Deal requests permission to:
- Read your Pipedrive contacts (to find existing matches)
- Create and update contacts (to add new people and attach phone numbers)
- Read your deals and pipelines (to show deal information in the sidebar)

Your Pipedrive credentials are never stored in the extension. Authentication uses secure tokens that expire and refresh automatically.

## Attaching a Phone to an Existing Pipedrive Person

Sometimes a contact already exists in Pipedrive but doesn't have their WhatsApp number saved. For example, you might have created a contact from an email inquiry, and now they're reaching out via WhatsApp for the first time.

Instead of creating a duplicate record, you can attach the WhatsApp number to the existing person:

1. When the sidebar shows "No match found," click "Attach to Existing Person"
2. A search field appears
3. Type the person's name
4. Select the correct contact from the search results
5. Click "Attach"

Chat2Deal adds the WhatsApp number to that person's Pipedrive record with the "WhatsApp" label. The sidebar then shows the matched person card with all their details.

This prevents duplicate contacts and ensures all communication channels for a single person are linked together in Pipedrive. You maintain a single source of truth for each customer.
